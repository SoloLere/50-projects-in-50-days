### What I learned

I can add Icons to my css file by using the Unicode of the desired icon.

```css

  position: absolute;
  content: "\f075";
  font-family: "Font Awesome 5 free";
  font-size: 7rem;
  color: #2ecc71;
  z-index: 0;
  opacity: 0.2;
  top: 15px;
  left: 20px;
}
```
